# Sindiana Executive Dashboard - Shared Editable

## מה חדש
- עיצוב משודרג עם צבעים מובחנים יותר בכל הגרפים.
- חיבור ל-Google Sheets כדי שאנשים יוכלו לערוך והדשבורד יתעדכן אצל כולם.
- תאריך "נכון ל" מתעדכן לפי עמודת UPDATED_AT או לפי זמן הרענון האחרון.
- כפתור עריכה וכפתור שליחה ב-WhatsApp.

## איך לחבר לעריכה משותפת
1. פתח Google Sheets חדש.
2. ייבא אליו את הקובץ data_template.csv.
3. שתף הרשאת עריכה עם מי שצריך.
4. בגוגל שיטס: File > Share > Publish to web.
5. בחר את הגיליון ופורמט CSV.
6. העתק את קישור ה-CSV.
7. פתח את הדשבורד באתר GitHub Pages.
8. לחץ "חיבור נתונים משותף", הדבק את קישור ה-CSV ואת קישור העריכה של הגיליון, ולחץ שמירה.

חשוב: GitHub Pages הוא אתר סטטי, לכן העריכה המשותפת עצמה חייבת להיות דרך Google Sheets או מקור נתונים חיצוני.
